Windows Phone 8 Training Kit

The Windows Phone Training Kit includes a comprehensive set of technical content designed to help you learn how to build applications for Windows Phone 8. The kit includes hands-on labs, presentations, demos, and code samples for every aspect of developing applications for Windows Phone 8. You can use the resources in the training kit to teach yourself or train others.

What's new in Windows Phone

Windows Phone 8 brings a number of new and updated features to app developers, including native gaming development, a phone-specific version of the Windows Runtime, and a new kernel.
